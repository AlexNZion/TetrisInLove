# TetrisInLove
Tetris clone made in the Love2D framework

Instructions:

RIGHT ARROW - Go right with the current piece
LEFT ARROW - Go left with the current piece
UP ARROW - Rotate piece clockwise
P - Pause game
R - Restart game
